# Langkah-langkah #
1. Clone repository
2. Buat sebuah database dengan nama "clinicease"
3. Import "clinicease.sql" ke dalam database
4. Buka browser(Chrome/firefox/edge)
5. Masukkan "http://localhost/UAS-PWEB-CLINICEASE"
